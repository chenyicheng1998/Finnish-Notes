# Omistus ja Kellonajat - 拥有和时间表达 ⏰

## 拥有结构 (minulla on)

芬兰语用 **-lla/-llä + on** 结构表达"拥有"（字面意思是"在我这里有"）。

### 肯定形式

| 人称       | 结构    | 中文    |
| ---------- | ------- | ------- |
| minulla on | 我有    | 我有    |
| sinulla on | 你有    | 你有    |
| hänellä on | 他/她有 | 他/她有 |
| meillä on  | 我们有  | 我们有  |
| teillä on  | 你们有  | 你们有  |
| heillä on  | 他们有  | 他们有  |

### 否定形式 (-lla/-llä + ei ole + 部分格)

| 人称           | 结构      | 中文      |
| -------------- | --------- | --------- |
| minulla ei ole | 我没有    | 我没有    |
| sinulla ei ole | 你没有    | 你没有    |
| hänellä ei ole | 他/她没有 | 他/她没有 |
| meillä ei ole  | 我们没有  | 我们没有  |
| teillä ei ole  | 你们没有  | 你们没有  |
| heillä ei ole  | 他们没有  | 他们没有  |

**注意：**否定句中，宾语要变为部分格。

### 常用表达

| 芬兰语           | 中文   |
| ---------------- | ------ |
| Minulla on nälkä | 我饿了 |
| Minulla on jano  | 我渴了 |
| Minulla on kiire | 我着急 |
| Minulla on kuuma | 我热   |
| Minulla on kylmä | 我冷   |

### 例句
- Minulla on auto. (我有车。)
- Minulla ei ole autoa. (我没有车。)
- Hänellä on kaksi lasta. (他/她有两个孩子。)

## 时间表达 (Mitä kello on?)

### 询问时间

**Mitä kello on?** = 现在几点？

### 整点

**Kello on [数字]** = 现在是[数字]点

- Kello on yksi. (一点。)
- Kello on kaksi. (两点。)
- Kello on kolme. (三点。)

### 刻钟和半点

| 表达           | 含义   | 例子                                 |
| -------------- | ------ | ------------------------------------ |
| varttia yli    | 过一刻 | Kello on varttia yli neljä (4:15)    |
| puoli          | 半     | Kello on puoli viisi (4:30)          |
| varttia vaille | 差一刻 | Kello on varttia vaille viisi (4:45) |

**重要：**"Puoli viisi" 是 4:30，不是 5:30！意思是"向着五点的一半"。

### 关键词

| 芬兰语  | 中文 |
| ------- | ---- |
| yli     | 超过 |
| vaille  | 差   |
| puoli   | 半   |
| varttia | 一刻 |

### 时间表达（在几点？）

| 整点              | 在该时刻       |
| ----------------- | -------------- |
| kello yksi        | yhdeltä        |
| kello kaksi       | kahdelta       |
| kello kolme       | kolmelta       |
| kello neljä       | neljältä       |
| kello viisi       | viideltä       |
| kello kuusi       | kuudelta       |
| kello seitsemän   | seitsemältä    |
| kello kahdeksan   | kahdeksalta    |
| kello yhdeksän    | yhdeksältä     |
| kello kymmenen    | kymmeneltä     |
| kello yksitoista  | yhdeltätoista  |
| kello kaksitoista | kahdeltatoista |

### 例句
- Kello on puoli seitsemän. (现在是六点半。)
- Tapaamme kuudelta. (我们六点见面。)
- Bussi lähtee viideltä. (公交车五点出发。)
