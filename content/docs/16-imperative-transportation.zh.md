# Käskymuoto ja Kulkuvälineet - 祈使句和交通方式 🚌

## 祈使语气 (Käskymuoto)

祈使句用于发出命令或指示。

### 构成（sinä人称）

| 动词类型 | 原形 | 词干（第一人称） | 肯定祈使句 | 否定祈使句 |
|-----------|------------|-------------------|---------------------|---------------------|
| 类型1 | lukea（读） | luen | Lue! | Älä lue! |
| 类型2 | syödä（吃） | syön | Syö! | Älä syö! |
| 类型3 | tulla（来） | tulen | Tule! | Älä tule! |
| 类型4 | avata（打开） | avaan | Avaa! | Älä avaa! |
| 类型5 | valita（选择） | valitsen | Valitse! | Älä valitse! |

**规则：**祈使句由第一人称单数词干（去掉-n）构成。

### 祈使句中的宾语格

1. **可数名词单数：**用原型
   - Avaa ovi. (打开门。)
   - Älä avaa ovea. (不要打开门。) - 否定句用部分格

2. **不可数名词：**用部分格
   - Juo vettä. (喝水。)
   - Älä juo vettä. (不要喝水。)

3. **部分格动词：**宾语用部分格
   - Auta minua. (帮我。)
   - Älä auta minua. (不要帮我。)

4. **天生复数的宾语：**用复数主格
   - Pese silmälasit. (洗眼镜。)
   - Älä pese silmälaseja. (不要洗眼镜。) - 否定句用部分格

### 常用祈使句

- Lue sivu 35! (读第35页！)
- Älä lue ruokapöydässä! (不要在餐桌上读书！)
- Syö hitaasti! (慢慢吃！)
- Älä syö sokeria! (不要吃糖！)
- Tule kotiin! (回家！)
- Älä tule vielä! (还不要来！)
- Avaa ovi! (开门！)
- Älä avaa tietokonetta! (不要打开电脑！)

## 交通方式

### 用 -lla/-llä 表示（乘坐）

| 芬兰语 | 中文 |
|---------|---------|
| autolla | 开车 |
| polkupyörällä | 骑自行车 |
| junalla | 坐火车 |
| lentokoneella | 坐飞机 |
| taksilla | 打车 |
| bussilla | 坐公交 |
| metrolla | 坐地铁 |
| laivalla | 坐船 |

**注意：**所格（-lla/-llä）用于表示交通工具。

### 移动动词

| 动词 | 中文 |
|------|---------|
| mennä | 去 |
| ajaa | 驾驶 |
| matkustaa | 旅行 |
| lähteä | 离开 |
| saapua | 到达 |
| tulla | 来 |

### 步行

**特殊情况：**
- Minä menen/tulen/lähden **kävellen**. (我走路去/来/离开。)

"kävellen"（走路）这个词不使用 -lla/-llä 格式。

### 例句

- Menen töihin autolla. (我开车去上班。)
- Hän matkustaa Tampereelle junalla. (他/她坐火车去坦佩雷。)
- Lapset tulevat kotiin kävellen. (孩子们走路回家。)
- Lähden lentokentälle taksilla. (我打车去机场。)
- Ajan polkupyörällä. (我骑自行车。)

### 在街上/附近

所格（-lla/-llä）也用于：
- 在街上
- 在附近或周边
- 使用工具

例句：
- Asun Mannerheimintie**llä**. (我住在曼纳海姆街。)
- Olen kadulla. (我在街上。)
- Soitan pianolla. (我弹钢琴。) - 使用乐器
