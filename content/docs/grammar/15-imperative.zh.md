# Käskymuoto - 祈使句 📢

祈使句用于发出命令、指示或提出请求。

## 构成（sinä 人称）

| 动词类型 | 不定式        | 词干（第一人称） | 肯定祈使句 | 否定祈使句   |
| -------- | ------------- | ---------------- | ---------- | ------------ |
| 类型1    | lukea (读)    | luen             | Lue!       | Älä lue!     |
| 类型2    | syödä (吃)    | syön             | Syö!       | Älä syö!     |
| 类型3    | tulla (来)    | tulen            | Tule!      | Älä tule!    |
| 类型4    | avata (打开)  | avaan            | Avaa!      | Älä avaa!    |
| 类型5    | valita (选择) | valitsen         | Valitse!   | Älä valitse! |

**构成规则：** 祈使句由第一人称单数词干（去掉 -n）构成。

## 祈使句中的宾语格

### 1. 可数单数名词
- **肯定句：** 使用主格
  - Avaa ovi. (打开门。)
  - Ota kirja. (拿书。)

- **否定句：** 使用部分格
  - Älä avaa ovea. (别打开门。)
  - Älä ota kirjaa. (别拿书。)

### 2. 不可数名词
- 总是使用部分格
  - Juo vettä. (喝水。)
  - Älä juo vettä. (别喝水。)

### 3. 部分格动词
- 宾语总是用部分格
  - Auta minua. (帮助我。)
  - Älä auta minua. (别帮助我。)

### 4. 天然复数宾语
- **肯定句：** 使用复数主格
  - Pese silmälasit. (洗眼镜。)

- **否定句：** 使用部分格
  - Älä pese silmälaseja. (别洗眼镜。)

## 常见例句

### 日常命令
- Lue sivu 35! (读第35页！)
- Älä lue ruokapöydässä! (别在饭桌上看书！)
- Syö hitaasti! (慢慢吃！)
- Älä syö sokeria! (别吃糖！)
- Juo kahvia. (喝咖啡。)
- Älä juo liikaa! (别喝太多！)

### 移动命令
- Tule kotiin! (回家！)
- Älä tule vielä! (还别来！)
- Mene ulos! (出去！)
- Älä mene sinne! (别去那里！)

### 动作命令
- Avaa ikkuna! (打开窗户！)
- Älä avaa tietokonetta! (别打开电脑！)
- Sulje ovi! (关门！)
- Älä sulje ikkunaa! (别关窗户！)

## 礼貌请求

要使祈使句更礼貌，可以添加：
- **ole hyvä** (请 / 字面意思"好好的")
- **kiitos** (请 / 字面意思"谢谢")

例句：
- Avaa ovi, ole hyvä. (请开门。)
- Tule sisään, kiitos. (请进来。)

## 复数祈使句（te 形式）

对于正式或复数命令，在词干后加 **-kaa/-kää**：
- Tulkaa sisään! (请进！- 复数/正式)
- Istukaa alas! (请坐！- 复数/正式)
- Älkää puhuko! (别说话！- 复数/正式)
