# Partitiivi - 部分格 📝

部分格是芬兰语中最重要的格之一。它表示部分数量或未完成的动作。

## 构成规则

1. **单元音结尾**（不含 e、部分 i）：加 **-a/-ä**
2. **以 -ia, -ea, -eä 结尾**：加 **-a/-ä**
3. **双元音或辅音字母结尾**：加 **-ta/-tä**
4. **字母 e 结尾**：加 **-tta/-ttä**
5. **以 -nen 结尾的词**：变为 **-sta/-stä**

## 何时使用部分格

### 1. 数字后面

**monta** + 部分格 = 许多（可数）

**vähän** + 部分格 = 一点

**paljon** + 部分格 = 许多（不可数）

**pari** + 部分格 = 一对/几个

**puoli** + 部分格 = 半

**nolla** + 部分格 = 零

例句：
- kaksi kirjaa (两本书)
- kolme taloa (三座房子)
- paljon vettä (很多水)
- vähän aikaa (一点时间)

### 2. 部分格动词后面

某些动词的宾语总是用部分格：

| 动词     | 中文      |
| -------- | --------- |
| auttaa   | 帮助      |
| etsiä    | 寻找      |
| katsoa   | 看        |
| puhua    | 说        |
| rakastaa | 爱        |
| soittaa  | 演奏      |
| ymmärtää | 明白/懂得 |

例句：
- Minä puhun suomea. (我说芬兰语。)
- Hän rakastaa musiikkia. (他/她爱音乐。)
- Katson televisiota. (我看电视。)

### 3. 否定句中

否定句中的宾语用部分格：

- Minulla ei ole autoa. (我没有车。)
- En ymmärrä suomea. (我不懂芬兰语。)
- En näe sinua. (我看不见你。)

### 4. 部分数量

当指不确定或部分数量时：

- Haluan kahvia. (我想要咖啡。) - 一些咖啡，不是特定的一杯
- Ostin maitoa. (我买了牛奶。) - 一些牛奶，不是全部牛奶

## 主格 vs. 部分格

比较这些例子：

- Kahvi on pöydällä. (咖啡在桌子上。) - 主格，特定物品
- Haluan kahvia. (我想要咖啡。) - 部分格，部分数量
- Syön omenan. (我吃了苹果。) - 宾格（单数时与主格相同），完整对象
- Syön omenaa. (我在吃苹果。) - 部分格，进行中/部分动作
