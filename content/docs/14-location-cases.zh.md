# Paikkasijakäännökset - 方位格 📍

芬兰语有六个方位格，分为两个系统：内部（里面）和外部（表面上）。

## 内部方位系统（S格）

### 入格 (Illatiivi) - Mihin?（去哪里？）

**词尾：** -Vn, -hVn, -seen

表示进入某物内部。

**构成规则：**
1. 非单音节词，以单元音结尾：双写最后一个元音 + n
2. 非单音节词，以两个不同元音结尾：双写最后一个元音 + n
3. 单音节词，以两个元音结尾：+hVn（V = 最后一个元音）
4. 非单音节词，以两个相同元音结尾：+seen

**例句：**
- Kissa menee laatikkoon. (猫进盒子里。)
- Kissa menee huoneeseen. (猫进房间里。)
- Menen kouluun. (我去学校。)

### 内格 (Inessiivi) - Missä?（在哪里？）

**词尾：** -ssa/-ssä

表示在某物内部。

**例句：**
- Kissa on laatikossa. (猫在盒子里。)
- Kissa on keittiössä. (猫在厨房里。)
- Kissa on vessassa. (猫在厕所里。)
- Minä nukun sängyssä. (我在床上睡觉。)

### 出格 (Elatiivi) - Mistä?（从哪里来？）

**词尾：** -sta/-stä

表示从某物内部出来。

**例句：**
- Kissa tulee laatikosta. (猫从盒子里来。)
- Kissa tulee huoneesta. (猫从房间里来。)
- Tulen kaupasta. (我从商店来。)

## 外部方位系统（L格）

### 向格 (Allatiivi) - Mihin?（去哪里？）

**词尾：** -lle

表示移动到某个表面或朝向某物。

**例句：**
- Kissa menee pöydälle. (猫去桌子上。)
- Kissa menee pihalle. (猫去院子。)
- Kissa menee kattolle. (猫去房顶上。)

### 所格 (Adessiivi) - Missä?（在哪里？）

**词尾：** -lla/-llä

表示在某个表面上、在某个地点或附近。

**例句：**
- Kissa on pöydällä. (猫在桌子上。)
- Kissa on lattialla. (猫在地上。)
- Kissa on tuolilla. (猫在椅子上。)
- Olen Helsingissä, Esplanadin puistolla. (我在赫尔辛基，在埃斯普拉纳迪公园。)

**注意：**也用于：
- 在街上
- 在某些地点附近
- 部分城市和地点

### 离格 (Ablatiivi) - Mistä?（从哪里来？）

**词尾：** -lta/-ltä

表示从某个表面离开。

**例句：**
- Kissa tulee pöydältä. (猫从桌子上离开。)
- Kissa tulee pihalta. (猫从院子离开。)
- Kissa tulee kattolta. (猫从屋顶离开。)

## 与动词的搭配

### 常用动词与方位格

**mennä** + Mihin = 去

**käydä** + Missä = 访问

**tulla** + Mistä = 来

## 存在表达

### 某处有某物

**肯定：** [地点 -ssa/-ssä 或 -lla/-llä] + on + [物品]

- Huoneessa on pöytä. (房间里有桌子。)
- Pöydällä on kirja. (桌子上有书。)

**否定：** [地点 -ssa/-ssä 或 -lla/-llä] + ei ole + [物品的部分格]

- Huoneessa ei ole pöytää. (房间里没有桌子。)
- Pöydällä ei ole kirjaa. (桌子上没有书。)

## 总结表

| 疑问词 | 内部（S） | 外部（L） |
|----------|-------------|-------------|
| 去哪里？(Mihin?) | 入格 -Vn/-hVn/-seen | 向格 -lle |
| 在哪里？(Missä?) | 内格 -ssa/-ssä | 所格 -lla/-llä |
| 从哪里来？(Mistä?) | 出格 -sta/-stä | 离格 -lta/-ltä |
