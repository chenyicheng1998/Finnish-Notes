# Omistus - 拥有表达 💰

## 拥有结构 (minulla on)

芬兰语用 **-lla/-llä + on** 结构来表达拥有（字面意思是"在我这里有"）。

这与英语不同，英语说"I have"，而芬兰语说"在我这里有"。

### 肯定形式

| 人称       | 结构    | 中文    |
| ---------- | ------- | ------- |
| minulla on | 我有    | 我有    |
| sinulla on | 你有    | 你有    |
| hänellä on | 他/她有 | 他/她有 |
| meillä on  | 我们有  | 我们有  |
| teillä on  | 你们有  | 你们有  |
| heillä on  | 他们有  | 他们有  |

### 否定形式 (-lla/-llä + ei ole + 部分格)

| 人称           | 结构      | 中文      |
| -------------- | --------- | --------- |
| minulla ei ole | 我没有    | 我没有    |
| sinulla ei ole | 你没有    | 你没有    |
| hänellä ei ole | 他/她没有 | 他/她没有 |
| meillä ei ole  | 我们没有  | 我们没有  |
| teillä ei ole  | 你们没有  | 你们没有  |
| heillä ei ole  | 他们没有  | 他们没有  |

**重要：** 否定句中，宾语要变为部分格。

### 例句

**肯定句：**
- Minulla on auto. (我有车。)
- Sinulla on koira. (你有狗。)
- Hänellä on kaksi lasta. (他/她有两个孩子。)
- Meillä on iso talo. (我们有大房子。)

**否定句：**
- Minulla ei ole autoa. (我没有车。)
- Sinulla ei ole kissaa. (你没有猫。)
- Hänellä ei ole aikaa. (他/她没有时间。)

### "minulla on" 的常用表达

| 芬兰语            | 中文                |
| ----------------- | ------------------- |
| Minulla on nälkä  | 我饿了              |
| Minulla on jano   | 我渴了              |
| Minulla on kiire  | 我赶时间            |
| Minulla on kuuma  | 我热                |
| Minulla on kylmä  | 我冷                |
| Minulla on ikävä  | 我想念（某人/某物） |
| Minulla on oikeus | 我有权利            |
| Minulla on lupa   | 我有许可            |

### 疑问句

- Onko sinulla autoa? (你有车吗？)
- Onko teillä lapsia? (你们有孩子吗？)
- Onko hänellä aikaa? (他/她有时间吗？)

### 其他拥有表达

**位置：**
- Hänellä on rahaa taskussa. (他/她口袋里有钱。)

**数量：**
- Minulla on paljon työtä. (我有很多工作。)
- Meillä on vähän aikaa. (我们时间不多。)
