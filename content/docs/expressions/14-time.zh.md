# Kellonajat - 时间表达 ⏰

## 询问时间

**Mitä kello on?** = 现在几点？

**Paljonko kello on?** = 现在几点？（另一种说法）

## 整点

**Kello on [数字]** = 现在是[数字]点

- Kello on yksi. (1点整 / 1:00)
- Kello on kaksi. (2点整 / 2:00)
- Kello on kolme. (3点整 / 3:00)
- Kello on neljä. (4点整 / 4:00)
- Kello on viisi. (5点整 / 5:00)

## 一刻钟和半小时

| 表达           | 含义   | 例子                                 |
| -------------- | ------ | ------------------------------------ |
| varttia yli    | 过一刻 | Kello on varttia yli neljä (4:15)    |
| puoli          | 半点   | Kello on puoli viisi (4:30)          |
| varttia vaille | 差一刻 | Kello on varttia vaille viisi (4:45) |

**重要：** "Puoli viisi" 意思是 4:30，不是 5:30！它表示"朝向五点的一半"。

## 关键词汇

| 芬兰语   | 中文    |
| -------- | ------- |
| yli      | 过/超过 |
| vaille   | 差/之前 |
| puoli    | 半      |
| varttia  | 一刻钟  |
| minuutti | 分钟    |

## 分钟表达

**过几分：**
- Kello on viisi yli kaksi. (2点过5分 / 2:05)
- Kello on kymmenen yli kolme. (3点过10分 / 3:10)

**差几分：**
- Kello on viisi vaille kolme. (3点差5分 / 2:55)
- Kello on kymmenen vaille neljä. (4点差10分 / 3:50)

## 时间点表达（在几点？）

使用离格 (-lta/-ltä) 来表达"在[时间]":

| 钟点时间          | 在该时间点     |
| ----------------- | -------------- |
| kello yksi        | yhdeltä        |
| kello kaksi       | kahdelta       |
| kello kolme       | kolmelta       |
| kello neljä       | neljältä       |
| kello viisi       | viideltä       |
| kello kuusi       | kuudelta       |
| kello seitsemän   | seitsemältä    |
| kello kahdeksan   | kahdeksalta    |
| kello yhdeksän    | yhdeksältä     |
| kello kymmenen    | kymmeneltä     |
| kello yksitoista  | yhdeltätoista  |
| kello kaksitoista | kahdeltatoista |

### 例句
- Koulu alkaa kahdeksalta. (学校8点开始。)
- Tapaan sinut viideltä. (我5点见你。)
- Bussi lähtee puoli seitsemältä. (公交车6点半发车。)

## 24小时制

在正式场合使用：
- Kello on kolmetoista. (13:00 / 下午1点)
- Kello on viisitoista kolmekymmentä. (15:30 / 下午3点半)
- Kello on kaksikymmentä. (20:00 / 晚上8点)
