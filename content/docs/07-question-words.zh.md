# Kysymyssanat - 疑问词 ❓

## 常用疑问词

| 中文          | 芬兰语         | 例句                      | 回答例句                   |
| ------------- | -------------- | ------------------------- | -------------------------- |
| 谁            | kuka           | Kuka sinä olet?           | Minä olen Olga.            |
| 什么 + be动词 | mikä           | Mikä päivä tänään on?     | Tänään on maanantai.       |
| 什么 + 动词   | mitä           | Mitä kieltä sinä puhut?   | Minä puhun italiaa.        |
| 什么时候      | milloin        | Milloin kurssi on?        | Kurssi on maanantaina.     |
| 哪国人        | minkämaalainen | Minkämaalainen Pedro on?  | Hän on brasilialainen.     |
| 在哪里        | missä          | Missä te asutte?          | Me asumme Suomessa.        |
| 从哪里        | mistä          | Mistä sinä olet kotoisin? | Olen kotoisin Brasiliasta. |
| 到哪里        | mihin          | Mihin sinä menet?         | Minä menen kotiin.         |
| 为什么        | miksi          | Miksi et osta jäätelöä?   | Koska minä säästän rahaa.  |
| 怎么样        | kuinka         | Kuinka vanha sinä olet?   | Minä olen 23 vuotta vanha. |
| 什么样的      | millainen      |                           |                            |

## 注释

### mikä vs mitä
- **mikä** - 与动词 "是" (olla) 一起使用
  - Mikä päivä tänään on? (今天是哪一天？)
- **mitä** - 与动作动词一起使用
  - Mitä sinä teet? (你在做什么？)

### 位置词
- **missä** - 在哪里（在某个位置）
- **mistä** - 从哪里（起源/来源）
- **mihin** - 到哪里（目的地）

## 例句

- **Kuka** hän on? (他/她是谁？)
- **Mikä** tämä on? (这是什么？)
- **Mitä** sinä syöt? (你在吃什么？)
- **Milloin** tapaamme? (我们什么时候见面？)
- **Missä** sinä asut? (你住在哪里？)
- **Mistä** sinä tulet? (你从哪里来？)
- **Mihin** sinä menet? (你要去哪里？)
- **Miksi** sinä opiskelet suomea? (你为什么学习芬兰语？)
- **Kuinka** voit? (你好吗？)
